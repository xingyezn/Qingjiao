## 📘 项目简介

“青椒之梯”是一个专为高校青年教师（“青椒”）打造的项目申报资料库，旨在提供**纵向科研项目**的申报指南与官方文件，帮助青年教师更高效地获取信息、规划申报、迈好科研第一步。

目前内容以**项目指南文件**为主，后续将逐步补充申报模板、撰写建议等实用资源。
